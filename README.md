# index-cn-lab

本仓库用于归档与发布多个独立 HTML 页面。这些页面不绑定特定域名或网站，更多是作为技术实验、内容展示或功能测试的集合。

## 项目简介

`index-cn-lab` 是一个轻量化的页面归档仓库，主要用于存放各类自包含的 HTML 文件。每个页面可独立运行，无需依赖外部服务或复杂后端。仓库内容不定期更新，供学习、参考或内部使用。

## 目录结构

```
.
├── pages/          # 存放各个 HTML 页面文件
├── assets/         # 静态资源（如 CSS、JS、图片等，可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，命名尽量体现其用途或主题。
- `assets/`：若页面需要共享资源，可放置于此，按需组织。
